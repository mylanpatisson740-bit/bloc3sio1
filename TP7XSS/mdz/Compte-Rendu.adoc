=== Question 4

**Question :**
Quels sont les caractères typiques utilisés lors d’une attaque XSS ?

**Réponse :**

Comme caractère typique il y a :

* les cheverons (< >) pour fermer les balises html, dont **<scrip>**
* la barre oblique (/) pour fermer une balise
* les guillemets (' ou ") pour mettre le reste d'une requête en commentaire
* le point virgule (;) pour fermer une instruction javascript
